License Pools
ImageGear Java PDF > Licensing > Runtime Licensing > Automatically Reported Runtime (Node-Locked) > License Pools

When Automatically Reported Runtime licenses are purchased, they are organized into pools of a specific version and platform for a particular toolkit. For example, if a customer developed two applications using ImageGear, one using Linux and the other using Windows DLL, they would purchase two pools of runtime licenses, one for the Linux platform and one for the Windows DLL platform. At installation, a new license would be pulled from the appropriate pool, depending upon which platform the application is using.
