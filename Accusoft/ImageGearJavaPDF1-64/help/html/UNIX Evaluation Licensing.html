Evaluation Licensing
ImageGear Java PDF > Licensing > Evaluation Licensing

During evaluation of Accusoft products, Evaluation Toolkit licenses can be used to try out products. However, the product will only function for a limited number of days from the activation of the Evaluation license.

No licensing calls are necessary to run the product in Evaluation mode.

Use the License Manager to obtain an Evaluation license for your computer.

When you determine the product is a good fit, you can purchase a Toolkit license to eliminate the trial timeout.
