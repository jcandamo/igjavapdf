ImageGear Java PDF
System Requirements
ImageGear Java PDF > Getting Started > System Requirements

Before installing ImageGear Java PDF, make sure your computer system meets the minimum requirements detailed in this section:

  • 64-bit Intel-based platform
  • GNU C Library version 2.14 or higher

    To check which version, if any, you currently have installed, run the following:

     
    Copy Code
    ldd --version
  • Java Development Kit (JDK): Oracle Java 8 or greater

    To check which version, if any, you currently have installed, run the following:

     
    Copy Code
    java -version
  • Maven 3 or greater

    To check which version, if any, you currently have installed, run the following:

     
    Copy Code
    mvn -version

Operating Systems

 ImageGear Java PDF has been tested on the following operating system(s):   

  • Linux (64-bit)
ImageGear Java PDF has been tested on the following operating systems: CentOS (64-bit) and Ubuntu (64-bit). ImageGear Java PDF should also run on other Linux distributions, as long as they fully satisfy the requirements listed in this section.
