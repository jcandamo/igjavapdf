Creating a PDF from an Image

To add an image to the PDF document the following steps could be performed:

  • Create a new instance of the Document class.
  • If you want to add an image to the existing document, use the openDocument method of Document class to load the PDF document.
  • Use the insertBlankPage method of Document class to add a new blank page to the PDF document.
  • Use the getPage method of Document class to retrieve the newly added or another specific page of the PDF document.
  • Create a new instance of AddImageOptions class and provide the necessary data to its members. In particular set the location and resulting size of the image to be added to the page. See the AddImageOptions class for details.
  • Use the addImageFromFile method of Page class to add an image from file to the PDF page, or use addImage method of Page class to add an image from a memory byte array.
  • Use the closePage of Page class to close page if you do not need it anymore.
  • Use the saveDocument method of Document class to save the resulting PDF document to a file.

 

The following is an illustration of how to create a new PDF document, add image from file to its first page and save it out to a PDF file:

 

import com.accusoft.imagegearpdf.*;

PDF pdf = new PDF();

// set license

pdf.setSolutionName("Accusoft Corporation");

pdf.setSolutionKey(0x12345678,0x9ABCDEF0,0x87654321,0x0FEDCBA9);

pdf.setOEMLicenseKey("2.0.WE...ff");

pdf.initialize();

Document doc = new Document();

 

// insert a new blank first page with default size

doc.insertBlankPage(0);

Page page = this.document.getPage(0);

 

AddImageOptions addOptions = new AddImageOptions();

options.setX(20);

options.setY(400);

options.setWidth(200);

options.setHeight(200);

 

page.addImageFromFile("image.png", addOptions);

 

page.closePage();

SaveOptions saveOptions = new SaveOptions();

doc.saveDocument("output.pdf", saveOptions);

doc.closeDocument();

pdf.terminate();

 

The following is an illustration of how to create a new PDF document, add image from a BufferedImage instance to its first page and save it out to a PDF file:

 

import com.accusoft.imagegearpdf.*;

import java.io.*;

import java.awt.image.BufferedImage;

import javax.imageio.ImageIO;

// NOTE: assume given BufferedImage instance

// BufferedImage bufferedImage;

PDF pdf = new PDF();

// set license

pdf.setSolutionName("Accusoft Corporation");

pdf.setSolutionKey(0x12345678,0x9ABCDEF0,0x87654321,0x0FEDCBA9);

pdf.setOEMLicenseKey("2.0.WE...ff");

pdf.initialize();

Document doc = new Document();

 

// insert a new blank first page with default size

doc.insertBlankPage(0);

 

Page page = this.document.getPage(0);

 

AddImageOptions addOptions = new AddImageOptions();

options.setX(20);

options.setY(400);

options.setWidth(200);

options.setHeight(200);

 

ByteArrayOutputStream outputStream = new ByteArrayOutputStream();

ImageIO.write(bufferedImage, "bmp", outputStream);

byte[] imageData = outputStream.toByteArray();

page.addImage(imageData, addOptions);

page.closePage();

 

SaveOptions saveOptions = new SaveOptions();

doc.saveDocument("output.pdf", saveOptions);

doc.closeDocument();

pdf.terminate();
